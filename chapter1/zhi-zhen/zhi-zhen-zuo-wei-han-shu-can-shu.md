1、空指针

* 地址0不能访问，指向地址0的指针就是空指针
* 可以用“NULL”关键字对任何类型的指针进行赋值。NULL实际上就是整数0，值为NULL的指针就是空指针。

  * eg：

  ```
  int * pn = NULL;
  char * pc = NULL;
  int * p2 = 0;    //0就是NULL
  ```
